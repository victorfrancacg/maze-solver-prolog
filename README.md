# MAZE SOLVER - PROLOG VERSION
Este é um projeto relacionado à disciplina "Paradigmas de Linguagens de Programação", ministrada pelo professor Ricardo Oliveira. 

Este projeto foi idealizado no início da disciplina, e foi anteriormente implementado utilizando o paradigma Funcional e a linguagem de programação Haskell. O desafio é, desta vez, implementá-lo com base nos pilares do paradigma Lógico e através da linguagem Prolog.

# MAZE SOLVER - PROLOG VERSION
## Este é um projeto relacionado à disciplina "Paradigmas de Linguagens de Programação", ministrada pelo professor Ricardo Oliveira. 

## Este repositório contém a tentativa de implementação do projeto (que foi idealizado no início da disciplina e que já foi previamente codificado utilizando o paradigma Funcional e a linguagem 'Haskell') em Prolog, utilizando as bases fundamentais do paradigma LÓGICO.
